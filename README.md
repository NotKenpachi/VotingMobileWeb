# YES or NO?

A mobile-friendly voting app that presents 100 thought-provoking dilemmas answered strictly with YES or NO.

## How to Run

1. Download or clone the project folder
2. Open `index.html` in any modern web browser
3. No server, no installs, no dependencies required

## File Structure

```
272MobileDev/
├── index.html   — app structure (three screens)
├── style.css    — dark theme, mobile-first styles
├── app.js       — 100 questions and all app logic
└── README.md    — this file
```

## Features

- 100 YES/NO dilemma questions
- Animated question transitions
- Progress bar and answer counter
- Results page grouped by YES / NO / Skipped
- Votes saved to LocalStorage (persists across sessions)
- Reset button to clear all votes
- Works on mobile and desktop

## How It Works

1. **Start** — tap "Start Voting" to begin (or resume where you left off)
2. **Vote** — tap YES or NO on each question; tap Skip to pass
3. **Results** — view all your answers grouped by response; reset anytime

## Browser Support

Works in any modern browser: Chrome, Firefox, Safari, Edge.
